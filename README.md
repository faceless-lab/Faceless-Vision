### Faceless Vision

##### About

Faceless Vision is an application for Face and Engagement detection.

##### Dependencies

The project requires

- [OpenCV](https://opencv.org/) - version 3.3.x and above
- [DLib](http://dlib.net/) - version 19.x and above 

##### Building

- `mkdir build`
- `cd build`
- `cmake ..`
- `make`
